角色設計的秘密

小測驗

請選擇適當的選項

 
  1. 在角色設計中，最重要的第一步通常是什麼？
    1.   決定角色的顏色
    2.   設定角色的背景故事
    3.   設計角色的服裝
    4.   先畫好角色的所有動作
  2. 若想讓角色更容易被記住，設計時最需要注意的是？
    1.   讓角色的細節越多越好
    2.   使用奇怪的動作
    3.   設計明顯且具辨識度的特徵
    4.   角色一定要穿鮮豔衣服
  3. 角色設計中的「輪廓（Silhouette）」重點是什麼？
    1.   角色即使只有外形，也能被一眼認出
    2.   角色的身高比
    3.   角色的影子顏色
    4.   角色的背景場景
  4. 若要設計一位讓人感覺溫暖、友好的角色，以下哪個選項最符合角色設計的原則？
    1.   尖銳的身形、深色調
    2.   柔和的線條、溫暖色系
    3.   怒眉、誇張表情
    4.   大量使用金屬材質
  5. 角色設計時加入「缺點」的主要目的為？
    1.   讓角色更容易畫
    2.   讓角色看起來更醜
    3.   讓觀眾討厭這個角色
    4.   增加故事張力，使角色更立體
